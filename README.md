# YouTube Video Finder — Chrome Extension

Auto-scrolls a YouTube channel's Videos page until it finds your bookmarked video.

## Installation (takes ~1 minute)

1. Open Chrome and go to: `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **"Load unpacked"**
4. Select this folder (`yt-scroller`)
5. The extension icon will appear in your toolbar ✓

## How to use

1. Go to a YouTube channel and click the **Videos** tab
2. Click the extension icon in your toolbar
3. Type part of your bookmarked video's title (doesn't need to be exact)
4. Click **Start Scrolling**
5. It will automatically scroll down, loading videos as it goes, until it finds a match — then highlights it in red and stops

## Tips

- You only need to type enough of the title to make it unique
- The extension works on partial matches, so "cooking ep 47" will match "Gordon's Cooking Show EP 47"
- If it reaches the end without finding it, double-check the title spelling
- You can click **Stop** at any time
